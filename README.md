# BookSearchComposeUi - Android App

A modern Android application built with Jetpack Compose that allows users to browse and search for books, with detailed information about each book. The app demonstrates clean architecture and modern Android development practices.

## ✨ Features

- **Book List & Search Screen**
    - Display a list of books with cover images
    - Search functionality to filter books by name
    - Loading and success states

- **Book Detail Screen**
    - Detailed book information (name, author, description, etc.)
    - High-resolution book cover
    - Smooth navigation from list to detail

## 🛠️ Tech Stack

- **Kotlin** - Primary programming language
- **Jetpack Compose** - Modern UI toolkit
- **Navigation Compose** - In-app navigation
- **Hilt** - Dependency injection
- **Coil** - Image loading
- **ViewModel** - UI state management
- **Coroutines** - Asynchronous programming
- **Material 3** - Material Design components

## 📋 Prerequisites

- Android Studio Otter 3 Feature Drop | 2025.2.3 or newer
- JDK 21
- Android SDK
- Minimum SDK level 24 (Android 7.0)
- Target SDK level 36 (Android 16)
